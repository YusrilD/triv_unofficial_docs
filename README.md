# Triv

## Project Overview
## This is not official document of triv

**Triv** is a mobile application for **crypto asset trading**, supporting both **spot** and **futures** markets. The application also provides wallet management and additional features to support user engagement in the crypto ecosystem. Triv is designed for users who want to manage and trade crypto assets through a mobile-first experience.

---

## Core Features (High-Level)

- **Spot trading** for crypto assets  
- **Futures trading** with leveraged instruments  
- **Wallet** feature for managing crypto balances  
- **Market** feature to view price movements and market data  
- **Staking** feature for earning rewards from supported assets  

Detailed trading logic, risk management rules, and wallet security mechanisms are not exposed.
